# MealOrderSystem

## 项目简介
基于安卓的网上订餐系统项目，项目开发完善中，是我在大学期间开发的，采用C/S(服务端/客户端)架构，Android网络通讯框架采用Volley，服务器开发采用JavaEE进行开发，通过HTTP协议进行网络数据的请求发送。

我的个人博客，http://blog.ittrading.cn，技术介绍的博客大都写在CSDN，个人博客有链接可以访问

## 开源协议
本项目遵循Apache2.0开源协议
